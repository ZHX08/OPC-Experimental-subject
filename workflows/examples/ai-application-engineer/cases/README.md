## 读法

建议按“JD → 简历 → 匹配评估 → 推荐报告 → feedback”顺序看。这样能直接观察：
- 什么样的证据能让 AI 应用工程师被推荐
- 什么样的缺口会导致失败样例被打回
- 人工反馈会怎样修正模型初稿
