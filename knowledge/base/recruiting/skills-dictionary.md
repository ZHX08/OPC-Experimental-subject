# 招聘技能词典（第一版）

## 前端工程师
### 核心关键词
- JavaScript
- TypeScript
- React
- Vue
- Vite
- Webpack
- 状态管理
- 组件化
- 性能优化
- Node.js

### 重点识别
- 是否独立负责过复杂模块
- 是否有线上问题处理经验
- 是否理解工程化和协作流程

### 核心关键词
- PRD
- 原型
- 流程图
- 优先级
- 指标
- 需求分析
- 版本规划
- 跨团队协作

## AI 应用工程师
### 核心关键词
- LLM
- Prompt
- RAG
- Agent
- 工作流
- 知识库
- Tool Calling
- Dify
- LangGraph
- 向量检索
- rerank
- 评测

- 是否做过真实业务场景
