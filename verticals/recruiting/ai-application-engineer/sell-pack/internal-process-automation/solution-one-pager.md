# 方案一页纸：内部流程自动化

## 1. 解决什么问题

把采购、法务、审批、运营里的重复整理、摘要、补字段和流转，做成一条可审计、可回滚、可监控的 AI 工作流。

## 2. 适用对象

- 采购
- 法务
- 运营
- 审批主管

## 3. 核心流程

1. 输入：表单、邮件、工单、文档
2. 解析：抽取结构化字段
3. 查询：拉取 system of record
4. 生成：输出摘要和建议
5. 确认：高风险字段人工确认
6. 回写：提交到 OA / ERP / 工单
7. 留痕：记录审计日志
8. 监控：观察失败率、延迟和回退率

## 4. 交付边界

- 自动化的是整理、归类、摘要和低风险流转
- 人工兜底的是高风险字段、异常情况和最终提交
- 关键系统必须可追踪、可回滚

## 5. 上线方式

- 先做 shadow mode
- 再做小范围试点
- 再开放到更多流程
- 每一步都保留人工确认点

## 6. 成功标准

- 时间更短
- 错误更少
- 流转更稳
- 审计更清楚
