# Recruiting MVP

## 目标
做一个能真实省时间的最小闭环。

## MVP 功能

### 1. JD 拆解
输入：JD 文本
输出：
- 核心要求
- 隐性要求
- 必须项 / 加分项
- 面试关注点
- 推荐搜索关键词

### 2. 简历评估
输入：候选人简历 + 目标 JD
输出：
- 匹配亮点
- 风险点
- 缺口项
- 建议是否推荐
- 跟进问题清单

### 3. 推荐报告生成
输入：候选人简历 + JD + 沟通纪要
输出：
- 候选人摘要
- 匹配理由
- 风险提示
- 推荐意见

## MVP 成功标准
- 单次 JD 拆解时间从 20 分钟压到 3 分钟内
- 推荐报告初稿可直接复用 60% 以上
- 人工修改后的最终稿可回流为新资产
