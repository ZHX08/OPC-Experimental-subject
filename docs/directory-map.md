## 说明
- `knowledge/base/`：行业固定知识
- `knowledge/cases/`：案例库
- `knowledge/glossary/`：术语表
- `templates/prompts/`：Prompt 模板
- `templates/reports/`：报告 / 方案模板
- `templates/sop/`：标准作业模板
- `workflows/definitions/`：工作流定义
- `workflows/examples/`：实际任务样例
- `feedback/reviews/`：人工修改与结果反馈
- `apps/internal-process-automation-pilot/`：第一版本地试点 UI
- `data/raw/`：原始材料
- `data/processed/`：清洗、切块后的数据
- `data/vectors/`：向量化产物或索引导出
