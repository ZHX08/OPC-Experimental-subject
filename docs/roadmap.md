# Roadmap

## Phase 0：定行业
- 从 1 个垂直行业开始
- 明确 1 个高频任务
- 明确目标用户是谁

## Phase 1：建知识底座
- 补术语表
- 补 FAQ
- 补案例
- 补模板
- 补 SOP

## Phase 2：打第一个工作流
- 定输入
- 定检索范围
- 定模板调用规则
- 定输出格式
- 定人工审核点

## Phase 3：建立反馈闭环
- 保存 AI 初稿
- 保存用户修改稿
- 保存最终稿
- 记录是否采用与原因

## Phase 4：产品化
- 从“能用”到“好用”
- 从单任务到多任务
- 从手工维护到半自动维护
- 从单人使用到团队协作
